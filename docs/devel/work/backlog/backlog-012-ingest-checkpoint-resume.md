## 概要

大規模な取込ジョブを進行中に中断・再開できるよう、進捗チェックポイントを永続化する。

## 補足

- 想定単位: DataSource からの URI / バイトオフセット / シーケンス番号など、
  DataSourcePlugin ごとに「再開キー」を抽象化する。
- リトライ時の冪等性（同一ドキュメントの再投入で重複インデックスにならないこと）を担保する。
